## First GitHub Issues To Create

Create these issues to show a realistic roadmap:

1. Add live plotting for serial data.
2. Add two-point voltage calibration.
3. Add multi-channel support for A0-A5.
4. Generate Markdown reports after each test run.
5. Add demo screenshots and a short hardware video.

## Good Commit Plan

If you want a clean commit history, use this order:

1. Initial Arduino firmware.
2. Add Python simulator and analyzer.
3. Add serial logging support.
4. Add unit tests and GitHub Actions.
5. Add documentation and sample data.
